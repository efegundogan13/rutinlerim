# Store İçin Gerekli Dosyalar

## ✅ HAZIR
- [x] AAB dosyası
- [x] Gizlilik Politikası
- [x] Açıklama metinleri
- [x] Kategori bilgisi

## ⚠️ GEREKLİ (Store yüklerken)

### 1. App Icon (512x512)
- Mevcut: assets/icon.png
- Store için: 512x512 px, PNG, şeffaf arka plan YOK

### 2. Feature Graphic (Google Play)
- Boyut: 1024 x 500 px
- Format: JPG veya PNG
- İçerik: App logosu + "Rutinlerim" yazısı

### 3. Ekran Görüntüleri (Screenshots)
**Minimum 2, maksimum 8 adet gerekli**

Önerilen ekranlar:
1. Ana ekran (döngü listesi)
2. Yeni döngü ekleme
3. Kategori filtreleme
4. Döngü detay
5. Bildirim örneği (opsiyonel)

**Boyut:** 1080 x 1920 px veya daha yüksek
**Format:** PNG veya JPG

### 4. Gizlilik Politikası URL
- Dosya hazır ama bir yere host etmeliyiz
- Seçenekler:
  a) GitHub'a yükle (ücretsiz)
  b) Basit bir web sitesi
  c) Google Sites (ücretsiz)

## 📱 Ekran Görüntülerini Nasıl Alacağız?

### Seçenek 1: Gerçek Telefon
- Uygulamayı Expo Go'da aç
- Ekran görüntüleri al
- 1080 x 1920'ye resize et

### Seçenek 2: Emulator
- Android Studio Emulator kullan
- Screenshots al
- Boyutlandır

### Seçenek 3: Mockup Oluştur (En Profesyonel)
- Figma veya Canva kullan
- Telefon mockup içinde ekran görüntüleri
- Daha güzel görünür

## 🚀 ŞİMDİ NE YAPMALIYIZ?

1. **Gizlilik Politikasını GitHub'a yükle** (2 dk)
2. **Ekran görüntüleri al** (10 dk)
3. **Feature graphic oluştur** (5 dk)
4. **Store'a yükle!**
